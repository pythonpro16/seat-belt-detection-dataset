This database contains images on the right side of the front window of the vehicle that are used to identify the driver's seat belt.
This file contains two folders.
•	The final_set belt folder contains the images of driving with a seat belt and without a seat belt. Using these images, it is randomly divided into training, validation, and test data in the seat_belt folder.
•	seat_belt folder contains training, validation and test data. Each of these files contains images of a driver with a seat belt and without a seat belt.
